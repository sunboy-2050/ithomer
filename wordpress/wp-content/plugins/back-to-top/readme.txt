Back To Top
Adds an "Back To Top" button to easily scroll long pages back to the top.

== Description ==
在页面的左侧或者右侧创建一个“返回顶部”的按钮，使用“返回顶部”按钮，可以方便快捷平滑的返回到blog的顶部。

This WordPress plugin adds an "Back To Top" button to easily scroll long pages back to the top.

== Installation ==

<h3>安装</h3>
<ol>
<li>下载并解压
<li>将此文件夹上传到Wordpress的插件目录：'Wordpress根目录/wp-content/plugins/'。</li>
<li>在后台的插件管理处激活。</li>
</ol>

<h3>Installation</h3>
<ol>
<li>download and unzip file
<li>Upload "back-to-top" folder into your plugins directory.</li>
<li>Activate the plugin.</li>
<li>Go to Appearance > Back To Top.</li>
<li>Configure settings to your needs.</li>
</ol>
